# KADO94

Excelで作っていた
- **94%シミュレーション（メイン）**
- **固定費カバー率（単月 + 期間集計）**
を Streamlit アプリに移植したものです。

## 使い方（ローカル）
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 使い方（STRIKEと同じ：GitHub + Streamlit Cloud）
1. GitHubで新規リポジトリ作成
2. `app.py` と `requirements.txt` をアップロード（コミット）
3. Streamlit Cloudで「New app」→そのリポジトリを選択→`app.py`を指定→Deploy

## 月次データ（期間集計）
「固定費カバー率・期間集計」タブの下部で、月次データ（CSV/Excel）をアップロードできます。

必須列：
- 年月（例: 2025-04-01）
- 稼働病床数
- 延べ患者数（人日）
- 入院収入（実績）

任意列：
- 入院単価（空欄なら自動計算）

※ 後方互換として、旧名称（`DPC入院収入（実績）`、`日当`）でも読み込みできます。


## グラフ
- 単月：実績 vs 目標（稼働率 / 入院収入 / 追加必要量）
- 期間：月次推移（稼働率・入院収入・増収額・固定費カバー率）


## 計算モード
- **高精度**：途中の丸めなし（理論値に近い）
- **Excel互換**：途中計算を円・人日単位で丸め（Excelの表示結果に合わせやすい）


## グラフ
グラフは Plotly を使用しています（見やすさ優先）。
